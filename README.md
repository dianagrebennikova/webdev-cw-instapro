# Instapro

MVP социальной сети для обмена фотографиями

## Первоначальная оценка

40 часов

## Фактически затраченное время

73 часа
В проекте реализовано:
Авторизация и регистрация
* Регистрация нового пользователя (логин, пароль, аватар).
* Авторизация зарегистрированного пользователя.
* Сохранение состояния авторизации после перезагрузки страницы.
* Выход из аккаунта.
Лента постов
* Загрузка ленты постов из API.
* Динамическая отрисовка постов.
* Отображение автора, даты, описания, изображения, лайков.
* Кликом по имени — переход на страницу конкретного пользователя.
* Доступна как авторизованным, так и неавторизованным пользователям.
Лента пользователя
* Загрузка постов конкретного пользователя.
* Страница выглядит и работает так же, как общая лента.
* Доступна всем (авторизованным и нет).
Лайки
* Авторизованный пользователь может лайкать и снимать лайк с поста.
* Лайк обновляется без перезагрузки страницы.
* Корректный счётчик лайков.
* Замена иконки лайка в зависимости от состояния.
* Синхронизация состояния с API.
Добавление поста
* Полностью рабочая страница добавления поста.
* Загрузка изображения через встроенный компонент.
* Поле ввода описания.
* Отправка поста в API.
* Автоматическое обновление ленты после успешного добавления.
* Проверка обязательности текста и изображения.
Роутинг и навигация
* Навигация между:
    * Лентой постов
    * Лентой пользователя
    * Авторизацией
    * Регистрацией
    * Добавлением поста
* SPA без перезагрузок.
Сборка и структура проекта
* Webpack со сборкой модулей.
* Разделение на компоненты: страницы, API, хедер, загрузчик изображений и т.д.
* Поддержка дев-режима и прод-режима.

Дополнительные фичи
1. Обработка ошибок с выводом сообщений
Обработаны ошибки API:
* неверный логин/пароль,
* занятый логин,
* пустые поля,
* ошибки при добавлении поста,
* неверная загрузка изображения.
2. Автоматическое обновление ленты после добавления поста
После отправки нового поста происходит обновление без перезагрузки
3. «Защита от XSS» 
Посты отображаются через безопасные элементы → HTML-теги не исполняются.
4. Адаптация под неавторизованного пользователя
Лайки отключены, кнопка добавления поста — недоступна.
5. Анимированные лайки.
При клике на кнопку «лайк» анимация плавно увеличивает и уменьшает иконку
6. Структурированная сборка Webpack
С поддержкой:
* HtmlWebpackPlugin
* DevServer
* разделением production/development режимов
